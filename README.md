# Customer Segmentation Analysis

## Project Overview
This project performs customer segmentation using K-Means Clustering to group customers based on their annual income and spending score.

## Objectives
- Analyze customer purchasing behavior
- Perform data preprocessing
- Perform Exploratory Data Analysis (EDA)
- Apply K-Means Clustering
- Visualize customer segments
- Store processed data in SQLite database

## Dataset
Mall Customers Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SQLite
- Jupyter Notebook

## Results
- Identified customer segments using K-Means Clustering.
- Visualized customer groups.
- Exported processed data into SQLite.
